# PlaceMate

A comprehensive application with frontend, backend, and AI capabilities.

## Project Structure

```
PlaceMate/
├── frontend/          # Next.js frontend application
├── backend/           # Backend API server
├── ai-engine/         # AI/ML processing engine
└── README.md          # This file
```

## Getting Started

### Frontend
Navigate to the `frontend` directory and follow the instructions in its README.

```bash
cd frontend
pnpm install
pnpm dev
```

### Backend
Navigate to the `backend` directory for backend setup instructions.

```bash
cd backend
# Follow backend README for setup
```

### AI Engine
Navigate to the `ai-engine` directory for AI engine setup instructions.

```bash
cd ai-engine
# Follow AI engine README for setup
```

## Directory Overview

- **frontend/**: Contains the Next.js application with all UI components, pages, and client-side logic
- **backend/**: Backend server implementation (to be set up)
- **ai-engine/**: AI/ML processing and inference engine (to be set up)

## Development

Each subdirectory has its own README with specific setup and development instructions.
